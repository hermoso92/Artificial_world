# Reports Memory

Los reportes generados por el `ai-core` deben:
- incluir fecha y operación origen
- distinguir `verificado`, `demo`, `externo` o `roadmap`
- enlazar al artefacto analizado cuando exista
- ser breves y reutilizables para documentación o depuración

Formato sugerido:
- resumen ejecutivo
- evidencia observada
- riesgos o límites
- siguiente paso recomendado
