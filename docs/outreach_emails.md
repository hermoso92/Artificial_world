## Dónde encontrar estudios indies

- **itch.io** — Filtra por "indie", "simulation", "RPG"
- **Game Jams** — Ludum Dare, Global Game Jam, itch.io jams
- **Twitter/X** — Busca #gamedev #indiedev #indiegame
- **LinkedIn** — "indie game developer", "game designer"
