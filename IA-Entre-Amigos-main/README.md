# IA entre amigos

Workshop de inteligencia artificial de ingenieros para ingenieros. Pensado para compartir entre amigos y colegas del sector, con contenido actualizado a **marzo de 2026**.

Presentacion interactiva construida con [Astro](https://astro.build/), navegable con teclado, tactil o raton.

## Requisitos

- Node.js 18+
- npm

## Instalacion

```bash
git clone https://github.com/686f6c61/IA-Entre-Amigos.git
cd IA-Entre-Amigos
npm install
```

## Uso

```bash
# Desarrollo
npm run dev

# Build estatico
npm run build

# Preview del build
npm run preview
```
