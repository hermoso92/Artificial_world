# Reference: Thesis and Core Rules

> Condensed from [docs/VISION_CIVILIZACIONES_VIVAS.md](../../../docs/VISION_CIVILIZACIONES_VIVAS.md)

## Thesis

**Artificial World** = base para crear civilizaciones vivas con memoria, héroes, refugios y comunidades.

- **2D** = verdad estratégica del sistema
- **3D** = encarnación futura (no implementada hoy)

## Founder Flow

1. Usuario entra por landing
2. Elige `CivilizationSeed`
3. Nombra constructor y refugio fundador
4. Se crea: héroe, refugio, mundo inicial, comunidad fundadora, crónica fundacional
5. Usuario observa, habita y expande

**Mensaje corto:** *Empieza con un refugio. Elige una semilla. Mira nacer tu civilización.*

## Regla de Foco

Cada cambio debe responder:

**¿Fortalece o distrae de "crear un refugio inicial y ver nacer una civilización"?**

Si distrae → posponer.

## Qué Existe Hoy

| Categoría | Ejemplos |
|-----------|----------|
| **Real** | Motor Python 2D, agentes, memoria, persistencia, Modo Sombra |
| **Parcial** | Mundos ligeros con semilla, historia como crónica, comunidad inicial |
| **Demo** | Refugios jugables web, HeroRefuge, flujo landing→héroe→refugio |
| **No existe** | 3D runtime, integración Python/JS, civilizaciones completas, DobackSoft real |

## Qué No Prometer

- 3D como implementado
- Integración total Python/JS
- Civilizaciones completas (diplomacia, linajes, guerras profundas)
- DobackSoft comercial dentro del repo
