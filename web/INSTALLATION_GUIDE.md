## Web Browser Usage
Artificial World is designed to run directly in your web browser with zero installation required.

### Requirements
- A modern web browser with JavaScript enabled.
- Internet connection (for initial load).

### Supported Browsers
- Google Chrome (Recommended)
- Mozilla Firefox
- Apple Safari
- Microsoft Edge

### Steps
1. Open your browser.
2. Navigate to [artificialworld.es](https://artificialworld.es).
3. Start exploring!

## Progressive Web App (PWA)
You can install Artificial World as a native-like application on your device for offline access and a full-screen experience.

### Benefits
- Works offline.
- Launches from your home screen or desktop.
- No browser UI (address bar, tabs) for a more immersive experience.

### Chrome / Edge (Desktop)
1. Visit the website.
2. Look for the install icon (a screen with a down arrow) in the right side of the address bar.
3. Click "Install".

### Chrome (Mobile / Android)
1. Visit the website.
2. Tap the three-dot menu in the top right.
3. Select "Add to Home screen" or "Install app".

### Safari (iOS)
1. Visit the website.
2. Tap the Share button (square with an up arrow) at the bottom.
3. Scroll down and tap "Add to Home Screen".

### Firefox / Samsung Internet
Look for the home icon with a plus sign in the address bar or the "Add to Home screen" option in the browser menu.
