## Sobre Artificial World

**Artificial World** es un ecosistema de código abierto para experimentar con agentes autónomos, simulación determinista y sistemas de inteligencia artificial auditables. La plataforma combina un simulador 2D, una arena de minijuegos lógicos, herramientas de visualización y documentación científica completa.

**Sitio web:** [artificialworld.es](https://artificialworld.es)
**Repositorio:** [github.com/hermoso92/Artificial_world](https://github.com/hermoso92/Artificial_world)
**Licencia:** MIT (Open Source)

---

## Misión y Visión

### Misión
Democratizar el acceso a herramientas de simulación avanzadas y fomentar la investigación abierta en inteligencia artificial y sistemas autónomos.

### Visión
Convertirse en la plataforma de referencia para la experimentación con agentes autónomos, ofreciendo un entorno transparente, auditable y educativo para investigadores, desarrolladores y entusiastas de la IA.

## Características Principales

### 🌐 Simulador 2D Determinista
- Agentes autónomos con toma de decisiones basada en utilidad
- Refugios y héroes con memoria
- Telemetría en tiempo real (agentes activos, refugios, ticks)
- Control total: velocidad, semilla, pausa/reproducción
- Motor determinista (misma semilla = mismo resultado)

### 🎮 Arena de Minijuegos
- **3 en Raya:** IA con estrategia básica
- **Damas:** Juego completo con capturas y coronaciones
- **Ajedrez:** En desarrollo

### 🔥 FireSimulator
- Visualización de propagación de fuego en cuadrícula 2D
- Controles interactivos de velocidad y reset

### 📄 Documentación Científica
- Paper completo con metodología y arquitectura
- Casos de uso y ejemplos prácticos
- Guías de usuario y desarrollador

### 💻 Código 100% Abierto
- Frontend: React + Vite + Tailwind CSS
- Backend: Python (motor determinista)
- Licencia MIT
- Contribuciones bienvenidas

### 📱 Progressive Web App (PWA)
- Instalable en cualquier dispositivo
- Soporte offline
- Experiencia nativa

## FAQ para Prensa

### ¿Qué hace único a Artificial World?
Artificial World combina simulación determinista, transparencia total (código abierto) y una experiencia de usuario moderna (PWA). A diferencia de otras plataformas, todo el código es auditable y reproducible.

### ¿Quién puede usar Artificial World?
- **Investigadores:** Para experimentar con agentes autónomos
- **Desarrolladores:** Para aprender sobre simulación y IA
- **Educadores:** Para enseñar conceptos de IA de forma visual
- **Entusiastas:** Para explorar y jugar con agentes inteligentes

### ¿Es gratis?
Sí, Artificial World es 100% gratuito y de código abierto bajo licencia MIT.

### ¿Cómo se financia el proyecto?
Actualmente es un proyecto de código abierto sin ánimo de lucro. En el futuro, se considerarán modelos de financiación sostenibles (donaciones, sponsors, planes premium opcionales).

### ¿Puedo contribuir al proyecto?
¡Absolutamente! El proyecto está abierto a contribuciones de código, diseño, documentación y feedback. Ver [CONTRIBUTING.md](https://github.com/hermoso92/Artificial_world/blob/main/CONTRIBUTING.md).

### ¿Qué tecnologías usa?
- **Frontend:** React 18, Vite, Tailwind CSS, Framer Motion
- **Backend:** Python (motor determinista)
- **PWA:** Service Worker, Web App Manifest
- **Hosting:** Hostinger Horizons

### ¿Funciona en móviles?
Sí, Artificial World es 100% responsivo y funciona en móviles, tablets y escritorio. Además, se puede instalar como PWA para una experiencia nativa.

### ¿Qué significa "determinista"?
Un sistema determinista siempre produce el mismo resultado dada la misma entrada. En Artificial World, usar la misma semilla siempre genera el mismo mundo y comportamiento de agentes, lo que permite reproducibilidad científica.

### ¿Cuándo estará disponible el multijugador?
El multijugador está planificado para Q3 2025. Ver el [ROADMAP.md](./ROADMAP.md) para más detalles.

### ¿Habrá una app móvil nativa?
Sí, está planificada para Q4 2025. Mientras tanto, la PWA ofrece una experiencia similar a una app nativa.
