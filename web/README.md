
# Artificial World

![Artificial World Banner](/og-image.png)

## Project Overview
**Artificial World** is an open-source web platform designed for the experimentation, visualization, and auditing of autonomous agents in deterministic environments. It combines 2D simulation, logical minigames, and telemetry tools in a modern Progressive Web App (PWA), allowing researchers, developers, and educators to explore emergent AI behavior transparently and reproducibly.

## Features
- **2D Simulation Engine:** Deterministic simulation of autonomous agents optimizing utility functions.
- **Arena Games:** Classic logical games (Tic-Tac-Toe, Checkers) to test AI decision-making.
- **FireSimulator:** 2D propagation simulation with strict physical rules.
- **Documentation:** Comprehensive scientific paper and architecture documentation included.
- **Open Source:** 100% open-source codebase (MIT License).
- **Accessibility:** WCAG AA compliant, keyboard navigable, and screen-reader friendly.
- **PWA Support:** Installable on desktop and mobile with offline capabilities.
