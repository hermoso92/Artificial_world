
# Guía de Usuario - Artificial World

Bienvenido a **Artificial World**, un ecosistema de simulación determinista y agentes autónomos. Esta guía te ayudará a explorar todas las funcionalidades de la plataforma.

## ¿Qué es Artificial World?

Artificial World es un proyecto de código abierto que combina simulación 2D, juegos lógicos y herramientas de visualización para explorar el comportamiento de agentes autónomos en entornos controlados. El proyecto se centra en la transparencia, la auditabilidad y la experimentación con sistemas deterministas.

**Características principales:**
- **Simulación 2D determinista** con agentes, refugios y memoria
- **Arena de minijuegos** (3 en Raya, Damas, Ajedrez en desarrollo)
- **FireSimulator** para visualización de propagación
- **Documentación científica** (Paper)
- **Código abierto** en GitHub

## Primeros Pasos

### 1. Página de Inicio (Landing Page)
Al visitar `artificialworld.es`, encontrarás:
- **Presentación del proyecto:** Visión, misión y características principales
- **Llamadas a la acción (CTAs):** Botones para explorar el Hub, jugar en la Arena o leer el Paper
- **Navegación clara:** Barra superior con enlaces a todas las secciones

### 2. Explorar el Hub Central
El **Hub** (`/hub`) es el directorio central de todas las superficies del proyecto:
- **7 tarjetas de superficie** organizadas en una cuadrícula responsiva
- Cada tarjeta muestra:
  - **Título** de la superficie
  - **Descripción** breve
  - **Insignia de estado** (REAL, DEMO, PARCIAL, ROADMAP)
  - **Botón de acción** para acceder o ver más información

## Navegación

### Barra de Navegación (Navbar)
La barra superior está siempre visible y contiene:
- **Logo:** Clic para volver a la página de inicio
- **Hub:** Acceso al directorio central
- **Arena:** Minijuegos lógicos
- **Simulador:** Constructor de mundos 2D
- **Paper:** Documentación científica
- **Código:** Enlace directo al repositorio de GitHub
