
# Arquitectura del Proyecto - Artificial World

Este documento describe la arquitectura técnica, estructura y decisiones de diseño del frontend de Artificial World.

## 1. Stack Tecnológico

- **Framework:** React 18 (Vite)
- **Enrutamiento:** React Router v6
- **Estilos:** Tailwind CSS v3
- **Componentes UI:** shadcn/ui (Radix UI + Tailwind)
- **Animaciones:** Framer Motion
- **Iconografía:** Lucide React
- **PWA:** Service Worker nativo + Web App Manifest
