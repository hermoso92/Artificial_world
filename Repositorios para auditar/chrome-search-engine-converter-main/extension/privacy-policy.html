Search Engine Converter - Política de privacidad

Versión 2.1.0 — Febrero 2026

Datos recopilados

Esta extensión no recopila, almacena ni transmite datos personales del usuario a ningún servidor externo.

Almacenamiento local

La extensión utiliza chrome.storage.local únicamente para guardar las preferencias de configuración del usuario (dominio de Amazon, visibilidad de motores, orden de botones). Estos datos permanecen en el navegador del usuario y nunca se envían a terceros.

Permisos

La extensión solicita los permisos mínimos necesarios: activeTab (acceso a la pestaña activa cuando el usuario interactúa), contextMenus (menús de clic derecho), storage (guardar preferencias).

Comunicaciones externas

La extensión no realiza ninguna petición de red. Toda la funcionalidad se ejecuta localmente en el navegador.
